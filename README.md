# 막차타 서버

## Description

막차타의 서버의 Cors 에러와 정제된 데이터를 내려주기 위한 서버입니다. <br/>
env키가 필요하시다면 요청해주시거나, 4조 피그잼에서 사용하시면 되겠습니다. <br/>
api 명세를 swagger를 사용해 정리해두었으니 필요하시면 확인하시면 되겠습니다! <br/>
명세 확인 ex) http://localhost:8080/api-docs  <br/>
